The sources of this repostory have been imported from http://lib.modeldriven.org/MDLibrary/trunk/Applications/Alf-Reference-Implementation/ using git-svn.

The branch tracking the SVN changes is called "modeldriven". To update the tracking branch type:
$ git checkout modeldriven
$ git svn rebase

If you want to merge the changes from the SVN into the master type:
$ git checkout master
$ git merge modeldriven
